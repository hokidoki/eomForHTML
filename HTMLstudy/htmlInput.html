input tag

HTML input 태그는 웹 기반양식에서 사용자의 데이터를 받을 수 있는 태그이다.
form 태그안에서 가장 중요한(most important) 역할을 하는 요소,
타입 속성에 따라서 다양한 방식으로 제공된다.
input tag에는 속성에 따라서 버튼을 만들 수도 있고, 텍스트, 이메일을 입력하는 창을 만들수도 있다.
로그인창을 만들때 필요한것은?(id 입력과 password 입력창 그리고 입력된 값을 보내는 버튼.)

name으로 묶이지 않은 radio tag : 중복 선택이 가능하다.
name으로 묶인 radio tag : name 그룹중 하나만 선택이 가능하다.

checkbox는 중복선택을 지원 해야하는 경우 사용되고,
radio 버튼은 그룹화하여 그룹 중 하나의 값만 선택해야하는 경우에 사용한다.
